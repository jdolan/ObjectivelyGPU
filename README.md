# ObjectivelyGPU

ObjectivelyGPU is a thin, idiomatic [Objectively](https://github.com/jdolan/Objectively)-style C wrapper around SDL3's [SDL_gpu](https://wiki.libsdl.org/SDL3/CategoryGPU) API.

## Dependencies

* [Objectively](https://github.com/jdolan/Objectively) >= 2.0.0
* [SDL3](https://github.com/libsdl-org/SDL) >= 3.2.0

## Building

    autoreconf -i
    ./configure
    make
    sudo make install
